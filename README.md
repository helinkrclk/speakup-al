# SpeakUp AI 🎤

## Problem
Üniversite öğrencilerinin sosyal kaygı ve mülakat korkusu nedeniyle kariyer fırsatlarını kaçırması. Topluluk önünde konuşurken yaşanan "donup kalma" sorununa çözüm sunuyoruz.

## Çözüm
Gemini API (Google AI Studio) kullanarak kullanıcıyı gerçekçi mülakat senaryolarıyla karşılaştıran, cevaplarını analiz eden ve anlık profesyonel geri bildirim veren akıllı bir asistan.

## Kullanılan Teknolojiler
* Gemini API (Google AI Studio)
* React / Vite
* Tailwind CSS
* Lovable (Deployment)

## Nasıl Çalıştırılır?
Uygulama Lovable üzerinden canlı olarak erişilebilirdir. Yerel çalıştırma için `npm install` ve `npm run dev` komutları kullanılır.
