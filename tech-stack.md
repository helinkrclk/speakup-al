# 🛠️ Tech Stack: Minimalist & Fast

Buildathon süresince hız her şeydir. Bu yüzden "No-Build / Low-Config" bir yapı seçildi:

| Katman | Teknoloji | Neden? |
| :--- | :--- | :--- |
| **Frontend** | HTML5, Tailwind CSS | Kurulum gerektirmez, CDN ile anında stil verilebilir. |
| **Logic** | Vanilla JavaScript (ES6+) | Framework karmaşasından kaçınıp saf performans sağlar. |
| **AI Engine** | Gemini 1.5 Flash API | Hızlı yanıt süresi ve geniş ücretsiz kota/uygun maliyet. |
| **Deployment** | Vercel / GitHub Pages | Saniyeler içinde yayına alınabilir. |
| **Storage** | Browser LocalStorage | Kullanıcı geçmişini veri tabanı kurmadan saklamak için.
